Для работы с программой необходимо передать 3 параметра:

0 - enc|dec (сжать|распаковать)
1 - путь+имя+расшиние входящего файла (пример - C:/Users/Misha/Desktop/huffman_coder_decoder/Тест22.txt)
2 - путь+имя+расшиние изходящего файла (пример - C:/Users/Misha/Desktop/huffman_coder_decoder/Тест22.hf)

Программа записывает таблицу частот непосредственно в закодированный файл, что упрощает последующую распаковку.
